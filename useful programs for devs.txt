This is a list of links to download all the programmes recommended for working on HGO. If you are not a developer, then just ignore this file. If for some reason you use a Mac or Linux, google the programs because the links provided are for Windows only.


- Github Desktop (used to keep HGO files the same across all developers, and also provide a convenient way to allow fans to download the mod) [REQUIRED] - https://central.github.com/deployments/desktop/desktop/latest/win32

- Notepad++ (used to more conveniently edit text files, and has way more features than standard notepad) [RECOMMENDED] - https://notepad.plus/download

- ImageMagick (used to make the bat files for resizing flags functional) [RECOMMENDED] - https://download.imagemagick.org/ImageMagick/download/binaries/ImageMagick-7.0.10-57-Q16-HDRI-x64-dll.exe

- Paint.Net (used to edit .dds and .bmp files, basically a free version of Photoshop) [RECOMMENDED] - https://www.dotpdn.com/files/paint.net.4.2.14.install.zip
- SageThumbs (program that allows you to see .tga and .dds files as thumbnails in Windows Explorer) [NOT REQUIRED WITH PAINT.NET, OTHERWISE RECOMMENDED] - https://sourceforge.net/projects/sagethumbs/files/latest/download

- 7zip (compressed file opener) [RECOMMENDED] - https://www.7-zip.org/a/7z1900-x64.exe
- WinRAR (alternative compressed file opener) [NOT REQUIRED WITH 7ZIP, OTHERWISE RECOMMENDED] - https://www.win-rar.com/postdownload.html?&L=0

- DropIt (used for automatically moving files from Paradox\HOI4\History\States to Paradox\HOI4\Mods\ahgoverhaul\history\states) [OPTIONAL] - https://sourceforge.net/projects/dropit/files/DropIt/v8.5.1/DropIt_v8.5.1_Setup.exe/download
